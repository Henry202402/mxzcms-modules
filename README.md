# mxzcms-modules

`mxzcms-modules` 是梦小记 CMS 的 Laravel 模块化管理包，用于统一模块的服务提供器、路由、配置、语言包、视图和迁移加载。

## 特性

- 自动注册已启用模块的 `AppServiceProvider`
- 自动加载模块 `web/api/admin` 路由
- 自动注册模块配置、语言包、视图和迁移
- 支持安装阶段的最小可用模块集回退

## 稳定性增强

当前版本针对以下场景做了额外兜底：

- 缓存为空或缓存结构异常时，不再在启动阶段直接报错
- 模块目录、路由文件、语言目录、配置文件缺失时自动跳过
- 安装控制器、跨域中间件、主程序统计类不存在时自动降级
- 安装中间件避免控制台和 `/install` 路由进入死循环跳转

## 说明

该包面向 `mxzcms` 主程序使用，依赖宿主项目已经提供 `module_path()` 等模块辅助函数。
